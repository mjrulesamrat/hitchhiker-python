## Get Started
You can add, subtract, multiply, divide numbers like this
```py
addition = 72 + 23 # 95
subtraction = 108 - 204 # -96
multiplication = 108 * 0.5 # 54
division = 108 / 9 # 12
exponentiation = 2 ** 3 # 8
modulo = 3 % 2 # 1
```
